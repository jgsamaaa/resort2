# Amara Ridge

A cinematic, multi-page luxury mountain resort concept set in the Philippine Cordillera. Built with Next.js 16, TypeScript, Tailwind CSS 4 and Framer Motion.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality checks

```bash
npm run lint
npm run build
```

## Routes

- `/` — resort overview
- `/stays` and `/stays/[slug]` — villa collection and six detail pages
- `/experiences`, `/wellness`, `/dining`, `/gallery` — resort discovery
- `/offers` — curated stay packages
- `/journal` and `/journal/[slug]` — editorial stories
- `/location`, `/about`, `/contact` — destination and brand information
- `/booking` — client-side stay inquiry flow

The resort, reviews, contact details and location are fictional. All visual assets were generated specifically for this concept with Philippine highland direction.
